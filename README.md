## 📢 프로젝트 소개
해당 시스템은 배달음식의 배달비를 여러 사람이 분담하는 시스템이다. 같은 위치의 음식점에서 주문을 하는 사람들끼리 배달비를 나누어, 갈수록 높아지는 배달비의 부담을 감소시키는 것을 목표로 한다. 그룹을 지어 동일한 음식점에서 주문을 한다하여 시스템의 이름은 Grouping이라 정했다.
